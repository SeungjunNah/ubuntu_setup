In Ubuntu 18.04, default gcc version is 7.
To build torch7 with CUDA 9.0, it should go with gcc 6.

also, replace line177 of "install-deps" as follows

# sudo apt-get install -y python-software-properties
sudo apt-get install -y software-properties-common

CC=gcc-6 CXX=gcc-6 TORCH_NVCC_FLAGS="-D__CUDA_NO_HALF_OPERATORS__" ./install.sh
